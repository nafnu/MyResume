About The Maze Project

The Maze Project is a Unity-based 3D maze game designed to provide an engaging and challenging experience for players. It was created to explore Unity mechanics, AI navigation, and 3D environment design.

Some of the challenges faced during development resulted in a simplified experience. Affected features include menu access and dynamic interactions, which were not implemented.



Features

  • Dynamic maze generation with scalable difficulty levels.
  • Player-controlled navigation with smooth camera movements.
  • Timer-based gameplay to challenge players' problem-solving skills.

This project began with the goal of integrating advanced AI and puzzle-solving mechanics within a wizard-themed maze. While development constraints led to a simplified final product, the game effectively demonstrates core Unity features, including player controls, camera integration, and dynamic timers.
