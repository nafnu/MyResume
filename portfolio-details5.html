Project Details

Basic Calculator - CI Pipeline Project

This project is a simple Python-based calculator that supports basic arithmetic operations like addition and subtraction.

It demonstrates Continuous Integration (CI) practices, including automated testing and CI/CD pipelines using Azure DevOps



Aspects important during the development:

  • Code Quality Assurance: Implementing branch protection rules ensured that all changes were reviewed for potential issues such as bugs, security vulnerabilities, or poor code quality.
  • Collaboration: The project encouraged teamwork and knowledge sharing, allowing team members to contribute by suggesting improvements, pointing out bugs, and identifying best practice.
  • Bug Prevention: Early detection of issues was facilitated by enforcing reviews before merging changes into the main branch.




Features

  • Addition: Add two number
  • Subtraction: Subtract two numbers.

It is important review compliance and accountability of the cloud provider: In regulated industries, enforcing reviews ensured a documented approval process.
