Bootstrap CSS and JS Integration

1. CSS Link for Bootstrap

  • Path: assets/vendor/bootstrap/css/bootstrap.min.css

    This includes the core Bootstrap styles.

2. Bootstrap JS Bundle

  • Path: assets/vendor/bootstrap/js/bootstrap.bundle.min.js

    This script enables Bootstrap's interactive components like modals, tooltips, and dropdowns.

3. Bootstrap Core Classes

Grid System

Classes like container, row, col-lg-4, gy-4, and others are used for layout and alignment.

Flex Utilities

Classes like d-flex, flex-column, justify-content-center are used for responsive layouts.

Spacing Utilities

Classes like p-3, mt-30, gy-4 manage margins, padding, and spacing.

Alignment Utilities

Classes like align-items-center, text-center align and position elements.

4. Bootstrap Buttons

Button classes include btn, btn-primary, and btn-secondary for styling buttons like "Download CV" or "Drop a message."

5. Bootstrap Navigation and Header

Navbar Features

Custom classes like navmenu, navicon, and active are used for responsive navigation.

Header Utilities

Classes like header and header-toggle are used for small-screen toggling (d-xl-none).

6. Bootstrap Icons

Icons include bi-house, bi-twitter-x, bi-envelope, and others for social links, contact details, and menus.

7. Modal Component

Modal classes like modal, modal-dialog, and btn-close are used for creating popups such as contact forms.

8. Bootstrap Form Classes

Form elements use classes like form-control, form-label, and mb-3 for input fields and labels.

9. Utilities for Scroll and Preloader

Scroll to Top

Classes like scroll-top and justify-content-center create a scroll-to-top button.

Loader/Preloader

The preloader class is used to indicate loading.

10. Summary of Components and Utilities

  • Grid System: Containers, Rows, Columns
  • Flexbox Utilities: Alignment, Flex properties
  • Buttons: btn-primary, btn-secondary
  • Icons: Bootstrap Icons
  • Modal: Full modal setup for contact form
  • Forms: Input styling with form-control
  • Spacing/Alignment: Margins, Padding, and Flexbox utilities
  • Scroll to Top: Scroll utility
  • Preloader: Loader utility
  • JS Bundle: Bootstrap interactive components

11. Miscellaneous Features

Portfolio Filtering

Isotope filtering dynamically toggles portfolio categories like Games, Pipelines, Websites.

Dynamic Animations

Animations are added using AOS (Animate On Scroll).

Custom Lightbox

Glightbox plugin enables image preview galleries.

Social Links Integration

Bootstrap icons enhance the display of social links.

Footer Design

Responsive footers are built using alignment utilities.
